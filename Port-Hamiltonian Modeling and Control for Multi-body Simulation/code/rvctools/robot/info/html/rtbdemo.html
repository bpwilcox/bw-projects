rtbdemo

Robot toolbox demonstrations

Displays popup menu of toolbox demonstration scripts that illustrate:

  • homogeneous transformations
  • trajectories
  • forward kinematics
  • inverse kinematics
  • robot animation
  • inverse dynamics
  • forward dynamics

Notes

  • The scripts require the user to periodically hit in order to move through the explanation.
  • Set PAUSE OFF if you want the scripts to run completely automatically.
